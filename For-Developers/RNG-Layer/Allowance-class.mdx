title: "Allowance Class"
description: "A utility class to verify USDT token permissions before interacting with the limited shared liquidity pool."
---

Before a user can deposit funds into the RNG Layer's liquidity pool, their smart account must grant permission to the protocol's smart contracts to move their USDT. The `Allowance` class provides dedicated read methods to verify these permissions. By checking allowances before attempting a deposit, you can build smoother user experiences. Prompting the user for an approval transaction only when it is strictly necessary helps you avoid failed transactions.

## Read Methods

The `Allowance` class provides read-only methods to check token allowances against specific smart accounts and contract addresses.

### `checkAllowance`
Checks the USDT token allowance for a smart account against the liquidity contract.

**Method Signature**
```typescript
checkAllowance(smartAccount: `0x${string}`): Promise<string>
```

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `smartAccount` | `0x${string}` | âś… | Smart account address |

**Returns**
`string` - Allowance amount converted from Wei (mwei / 6 decimals).

**Example**
```typescript
const allowanceInstance = new Allowance();

const allowance = await allowanceInstance.checkAllowance(
  "0xa682C9022871881b3257A72E103B50fC75013245"
);

console.log(allowance); // "0"
```

---

### `getAllowanceByAddress`
Checks the USDT token allowance for a smart account against a specific contract address.

**Method Signature**
```typescript
getAllowanceByAddress(
  smartAccount: `0x${string}`,
  contractAddress: `0x${string}`
): Promise<bigint>
```

**Parameters**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `smartAccount` | `0x${string}` | âś… | Smart account address |
| `contractAddress` | `0x${string}` | âś… | Contract address to check allowance for |

**Returns**
`bigint` - Raw allowance value (not converted).

**Example**
```typescript
const allowanceInstance = new Allowance();

const allowance = await allowanceInstance.getAllowanceByAddress(
  "0xa682C9022871881b3257A72E103B50fC75013245",
  "0xc973DE9a335887e86309eC8419E0430fF8C0293c"
);

console.log(allowance); // 9999999999999999999999999999996176830000n
```
