---
title: "Local Development Setup"
description: "Instructions for building and linking the Entropy Layer SDK locally."
---

To use this SDK locally during development:

Navigate to the SDK directory and link it:

```bash
cd rng-layer-sdk
yarn build
yarn link
```

In your consumer application directory, link to the local SDK:

```bash
cd /path/to/your/consumer/app
yarn link "rng-layer-sdk"
```

After making changes to the SDK, rebuild it:

```bash
yarn build
```

Restart your consumer application to pick up the changes.
